### 1. 集合管理（Collections）
```
卡片式展示，支持：

✅ 快速打开编辑器
✅ 一键运行测试
```

### 2. 请求编辑器（Request Editor）
```
Postman 风格布局：
✅ 左侧：集合和请求树形导航
✅ 中央：请求编辑面板
  - 请求方法选择
  - URL 输入
  - 发送按钮
✅ 右侧：标签页系统
  - Headers 编辑
  - Body 编辑
  - Params 编辑
  - Response 查看
```

### 3. 环境管理（Environment Manager）
```
✅ 创建多个环境（Dev/Test/Prod）
✅ 配置 Base URL
✅ 设置默认 Headers
✅ 环境切换时自动注入
✅ 支持变量定义（预留）
```

### 4. 响应查看（Response Viewer）
```
✅ 显示 HTTP 状态码
✅ 显示响应时间
✅ 格式化 JSON 显示
✅ Headers 详情展示
✅ 自动识别数据类型
```

### 5. 测试运行（Test Runner）
```
✅ 集成 Pytest 框架
✅ 执行整个集合的测试
✅ 生成 Allure 报告
✅ 报告历史列表
✅ 一键查看报告
```

### 6. 导入导出（Import/Export）
```
✅ 导入 Postman JSON 集合
✅ 导出为 Postman 格式
✅ 自动解析和转换
✅ 保持集合结构
```
